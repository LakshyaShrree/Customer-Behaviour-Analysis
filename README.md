This project focuses on analyzing customer shopping behaviour using an end-to-end data analytics workflow. Python was used for data cleaning and exploration, PostgreSQL for structured SQL analysis, and Power BI for interactive data visualization.

The goal of the project is to understand customer demographics, purchasing patterns, subscription impact, and revenue trends to support data-driven business decisions.
